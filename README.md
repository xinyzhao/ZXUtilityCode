ZXUtilityCode
===============
Provides a few utility codes for iOS.

## Requirements

* Requires iOS 7.0 or later
* Requires Automatic Reference Counting (ARC)

## Components

* AMRPlayer
* BDLocationManager
* Coordinates
* WKWebView+H5Pay
* ZXColorPickerController
